# Mini Python SIEM
A simple, beginner-friendly SIEM for demonstration and real-life learning.

## Features
- Log collection
- Parsing & normalization
- Threat detection
- Web dashboard (Flask)

## Usage
Run each component:

### 1. Start log generator
''' bash
python3 collector.py '''

### 2. Parse logs
''' bash
python3 parser.py '''

### 3. Analyze threats
''' bash
python3 analyzer.py '''

### 4. Start dashboard

Open browser at: http://127.0.0.1:5000
